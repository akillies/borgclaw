THE ASSIMILATOR
Plug in. Run one script. Machine joins the hive.
1

Detect Hardware

Script auto-detects OS (macOS/Linux/Windows), CPU architecture, RAM, GPU. Chooses the right model profile: heavy (32GB+ / GPU), standard (16GB+), or light (8GB+).

2

Install Ollama

One-line install. curl -fsSL https://ollama.com/install.sh | sh on Mac/Linux. Windows installer on the USB. Idempotent — skips if already installed.

3

Pull Right Model

Based on hardware profile: qwen2.5:14b for heavy nodes, qwen2.5:7b for standard, qwen2.5:3b for light. Always pulls nomic-embed-text (tiny, useful everywhere). Optional: pre-staged GGUF files on the USB skip the download.

4

Configure LAN Access

Sets OLLAMA_HOST=0.0.0.0 so the node is reachable from any machine on the home network. Restarts Ollama service.

5

Register with Gateway

Writes a borgclaw-node-registration.json with hostname, IP, port, profile, model, hardware specs. Future: auto-updates the nginx upstream config on the Mac Mini gateway.

6

Verify & Report

Runs a test inference call. Confirms the node responds. Prints the curl command to test from other machines. Outputs: "NODE ASSIMILATED."
